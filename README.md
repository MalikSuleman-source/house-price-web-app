# 🏠 EstimateAI — House Price Predictor Web App

A professional AI-powered web application that predicts 
house prices in real time using Machine Learning.

## 🌐 Features
- Beautiful dark luxury UI
- Real-time price prediction
- ML model trained on 20,000+ properties
- Built with Flask & Scikit-learn

## 🛠️ Tech Stack
Python | Flask | Scikit-learn | HTML | CSS | Pickle

## ▶️ How to Run
pip install flask scikit-learn pandas numpy
python model.py
python app.py
Then open: http://127.0.0.1:5000
